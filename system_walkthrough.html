System Overview

The KLD GradeVault system is architected into two distinct zones: a Public Zone for entry and verification, and a secure Private Zone tailored to specific permissions.

Public Zone

Accessible to anyone. Serves as the secure gateway for authentication and registration.

Private Zone

Restricted access based on roles: Admin, Teacher, and Student. Contains the core functionality.

Public Zone

Public

Login Portal

File: login.php

The gatekeeper of the system. Authenticates credentials and routes users to their specific dashboards. Blocks unverified or unapproved accounts.

Public

Registration Page

File: register.php

Smart form with dynamic dropdowns. Enforces security by requiring institutional emails and OTP verification. Teachers are placed in a "pending" state upon registration.

Public

Account Recovery

File: forgot_password.php

Allows users to securely reset their credentials via email link.

Admin Zone (Institute Head)

Admin

Admin Dashboard

File: admin_dashboard.php

High-level view of the institute's health. Provides quick access to management tools and system stats.

Admin

Faculty Management

File: admin_faculty.php

Critical workflow for approving teacher accounts. Admins must explicitly "Approve" pending teachers before they can access the system.

Admin

Student Registry

File: admin_students.php

Manage student enrollment statuses and view student lists within the institute.

Admin

Institute Settings

File: admin_settings.php

Configure global variables like academic years, semesters, and system maintenance switches.

Teacher Zone (Faculty)

Teacher

Teacher Dashboard

File: teacher_dashboard.php

Central hub for teachers. Lists assigned subjects and provides access to grading tools.

Teacher

Grade Import

File: grade_upload.php

Drag-and-drop interface for Excel files. Parses files instantly, validates Student IDs, and allows data preview before committing.

Teacher

Class Records

File: teacher_records.php

Digital ledger for manual grade management. Search for specific students and edit or delete grade entries.

Student Zone

Student

Student Dashboard

File: student_dashboard.php

Real-time report card. Displays recent grades, GPA, and visual indicators for pass/fail status.

Student

Academic History

File: student_history.php

Transcript-style timeline of grades across all years and semesters.

Student

Print Preview

File: print_grades.php

Formats current grades into a branded, official PDF layout suitable for printing.
